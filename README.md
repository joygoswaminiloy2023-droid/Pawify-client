## 🐾 About Pawify

Pawify is a full-stack **pet e-commerce marketplace** connecting pet parents with trusted vendors across Bangladesh. It offers same-day delivery in Dhaka, verified vendors, and a wide range of pet food, toys, and accessories — all wrapped in a smooth three-role platform experience.

## ✨ Features

| 🐶 Feature | Description |
|---|---|
| 🛍️ **Multi-Vendor Marketplace** | Verified vendors list and sell pet products directly |
| 🔐 **Secure Auth** | Better Auth with JWT/JWKS (`jose`) + Google OAuth |
| 👥 **Three-Role System** | User / Vendor / Admin, each with a dedicated dashboard |
| ✅ **Product Approval Workflow** | Pending → Admin Review → Public Listing |
| 💳 **Stripe Payments** | Webhook-free confirmation via polling |
| 📊 **Analytics Dashboards** | Recharts-powered insights for every role |
| 🚚 **Fast Delivery** | Same-day in Dhaka, 24–48h nationwide |

## 🐈 Tech Stack

<div align="center">

```
Frontend   →  Next.js
Backend    →  Express.js
Database   →  MongoDB Atlas
Auth       →  Better Auth · JWT/JWKS (jose) · Google OAuth
Payments   →  Stripe (polling-based confirmation)
Charts     →  Recharts
```
